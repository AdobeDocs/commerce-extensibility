# Custom attributes modules

The custom attributes modules (`magento/out-of-process-custom-attributes` and `magento/out-of-process-custom-attributes-b2b`) allow out-of-process services to attach and persist custom attributes on Commerce data models. They include REST and GraphQL support, as well as admin UI integration for order create/view flows. The B2B module extends this support to negotiable quotes and companies.

The base module is available on all platforms, but the B2B module is available only on Adobe Commerce as a Cloud Service.

For details on how to use these modules, see the following documentation:

* [REST](https://developer.adobe.com/commerce/webapi/rest/modules/custom-attributes/)
* [GraphQL](https://developer.adobe.com/commerce/webapi/graphql/schema/attributes/mutations/)
