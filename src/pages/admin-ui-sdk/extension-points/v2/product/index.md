# Product extension points (V2)

The Adobe Commerce Admin UI SDK enables you to make modifications to the following elements on the **Catalog** > **Products** page in the Adobe Commerce Admin:

* Add [custom columns](grid-columns.md) to the product grid.
* Add [custom mass actions](mass-action.md) to the product grid.
